## Keyboard Shortcuts 

This website works keyboard-only too  
- `Arrow keys` go back and forward 
- `P` go to a random page 
- `S` search
- `A` shuffle artworks
- `I` shuffle idea machine  

Standard browser keys are the same 
- `Tab` and `Shift+Tab` to move focus 
- `Home` go top 
- `End` go bottom 
- `Space` scroll down
