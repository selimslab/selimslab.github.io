## setup

1. clone the repo
3. install ruby <https://www.ruby-lang.org/en/downloads/>
4. `gem install bundler jekyll`
5. `bundle install`

## serve
```
bundle exec jekyll serve --trace
```

`--incremental`

`--profile` 

## Features 
1. Minimal
2. No ads
3. Wiki links
4. Backlinks
5. Works offline
6. Dark mode
7. Search
8. Roll a dice for pages, artworks, and ideas
9. Syntax highlighter
