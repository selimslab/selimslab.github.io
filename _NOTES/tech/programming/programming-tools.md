## Keyboard shortcuts

### Mac terminal

    Clear line: cmd U 


### VSCode 

    Multiple cursors: option + shift + click

    Go top: cmd ⬆️
    Go bottom: cmd ⬇️

    Zoom in: cmd shift 0 
    Zoom out: cmd - 

    Select next occurrence: cmd D
    Select all occurrences: cmd + shift + L

    Fold all: Ctrl + K + 0
    Unfold all: Ctrl + K + J

    Swith to explorer: cmd 0
