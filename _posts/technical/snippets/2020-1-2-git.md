## rebase

> no rebase or amend in public repositories


 `git stash`

 `git checkout branchname`

 `git rebase -i `

 --> make changes

 `git rebase continue`

 --> solve conflicts

 `git rebase continue`

`git rebase --drop `

## rebase on a historical commit 

 `git stash`

 -> checkout to the commit

 -> make changes

 -> amend commit 

 -> create new branch there

 -> checkout to master again

 `git rebase rebaser`

 -> solve conflicts 

 `git rebase continue`
