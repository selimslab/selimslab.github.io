## Unicode
a standard to encode text. 

It assigns unique code points to characters.   

5: U+0035    
A: U+0041

UTF: Unicode Transformation Format  
UTF-8: uses 8-bit blocks. one byte for ascii, up to 4 bytes for the rest.
